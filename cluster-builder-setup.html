Cluster Builder Desktop Setup

Cluster Builder requires VMware Workstation for Linux. VMware Workstation requires manual user download and installation. There is a 30-day free trial period.

Download

You will then need to setup your Github account so that the cluster builder repo can be fetched.

Eg. 

      # create a new keys
      ssh-keygen -f /home/admin/.ssh/id_rsa -t rsa -N ''

      # add this to your Github ssh keys
      cat ~/.ssh/id_rsa.pub

      ...
      # then
      mkdir -p ~/Workspace
      cd ~/Workspace
      git clone git@github.com:ids/cluster-builder.git
